    Browser Object Model

    The BOM properties and methods are made available through the window object.

    Global variables are actual properties of a global object. In a browser environment, the global object is the window object.

    Dialogs

    window.alert() will pause the execution of the program and display a message in a dialog box.

    window.confirm() method will stop the execution of the program and display a confirmation dialog. It returns a true or false boolean.

    window.prompt() method will stop the execution of the program and displays a dialog that shows a message provided as an argument. The user can input data and will be returned.

    window.location.href method returns the full URL as a string. It can also be changed by assignment (using the = sign). If this is done, the page will be reloaded using the new property.

    window.location.protocol returns a string describing the protocol used (such as http, https, pop2, ftp etc.).

    The host property returns a string describing the domain of the current URL and the port number.

    The hostname property returns a string describing the domain of the current URL.

    The port property returns a string describing the port number.

    The pathname property returns a string of the path that follows the domain.

    The origin property returns a string that shows the protocol and domain where the current page originated from.

    The window.location object also has the following methods:

    • The reload() method can be used to force a reload of the current page.
    • The assign() method can be used to load another resource from a URL provided as a parameter.
    • The replace() method is almost the same as the assign() method, except the current page will not be stored in the session history, so the user will be unable to navigate back to it using the back button.
    • The toString() method returns a string containing the whole URL.

    Controlling Windows

    • The window.open() method can be used to open a new window.
    • The .close() method can be used to close a window.
    • The window.moveTo(0,0) method takes two parameters that are the X and Y coordinates of the screen that the window is to be moved to.
    • The window.resizeTo(100,100) method takes two parameters that specify the width and height of the resized window’s dimensions.

    Document Object

    • The write('hello') method simply writes a string of text to the page. It can also be used within a document inside script tags to inject a string into the markup.This will not overwrite the rest of the HTML on the page.
    • The document.cookie property creates a cookie, which can be assigned to JavaScript’s “cookie jar” .Assigning another cookie to it won’ t overwrite the entire property, it will just append it to the end of the string .A cookie’ s value can be changed by reassigning it to document .cookie using the same name but a different value.To see the current contents of the cookie jar, simply enter document.cookie.
    • We can use the split method to break the string into an array containing each name / value pair, then use a for of loop to iterate through the array.

      • const cookies = document.cookie.split("; ");
      for (crumb of cookies) {
      const [key, value] = crumb.split("=");
      console.log(`The value of ${key} is ${value}`); }

    • Cookies can be made persistent― that is, lasting beyond the browser session― by adding "; expires=date" to the end of the cookie when it’ s set, where date is a date value in the UTC String format Day, DD - Mon - YYYY HH: MM: SS GMT.To remove a cookie, you need to set it to expire at a time in the past .

      • const expiryDate = new Date();
      const tomorrow = expiryDate.getTime() + 1000 * 60 * 60 * 24;
      expiryDate.setTime(tomorrow);

      document.cookie = `name=Batman; expires=${ expiryDate.toUTCString()}`;

    Timing Functions

    • The window.setTimeout() method accepts a callback to a function as its first parameter and a number of milliseconds as its second parameter.
    • The window.setInterval() method works in a similar way to window.setTimeout(), except that it will repeatedly invoke the callback function after every given number of milliseconds.
    • The setTimeOut() and setInterval() methods can be used to animate elements on a web page.

    The - data attribute

    The data - attribute is a way of embedding data in a web page using custom attributes that are ignored by the browser .They’ re also private to a page, so are not intended to be used by an external service– their sole purpose is to be used by a JavaScript program .

    The data - powers attribute can be accessed using the following code:


    Superman


    const superman = document.getElementById('hero');
    const powers = superman.dataset.powers;

    HTML5 APIs

    To save a value locally, use:


    localStorage.setItem('name', 'Walter White');



    Closing the browser will not remove the stored value:


    localStorage.getItem('name');



    To change its value, we reference it again:

    localStorage.name = 'Heisenberg';
    console.log(localStorage.name);



    To remove the local storage:

    localStorage.removeItem('name');



    To completely remove everything stored in local storage:

    localStorage.clear();
