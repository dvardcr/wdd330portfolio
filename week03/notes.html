    Object Methods

    Objects in JS are used to contain information about users, orders, etc. Any action is represented by function properties. A function that is a property of an object is called its method.

    This

    To access the object, a method can use the this keyword.

    this.value

    this is not bound which means that it can be used in any function, even if it’s not a method of an object.

    When you create a new instance of an object with the new keyword, this refers to the instance.

    Since this refers to the object, you can use methods to get the instance of an object.

    Bind

    bind is a method that is present in every function. It allows you to change the this context. This method takes in any number of arguments and returns the bound functio/n.

    All objects are mutable which means that its properties and methods can be changed or removed, and new properties and methods can be added to the object, even if it was declared using const.

    Creating Objects

    const spiderman = {};

    const name = 'Iron Man';
    const realName = 'Tony Stark';

    const ironMan = { name, realName };

    Accessing Properties

    Bracket notation has a few advantages: it's the only way to access nonstandard property and method names that don’t follow the variable naming rules. It also lets you evaluate an expression and use it as the property key.

    Computed Properties

    JavaScript code can be placed inside square brackets and the property key will be the return value of that code.

    Calling Methods

    To call an object’s method we can also use dot or bracket notation.

    Checking if Properties or Methods Exist

    The in operator can be used to check whether an object has a particular property.

    method in object

    hasOwnProperty()

    IMPORTANT: This method will only return any properties that belong to that particular object, whereas using in or !== undefined will return true, even if the property has been inherited from another object.

    We can loop through all of an object’s properties and methods by using a for in loop

    The Object.keys() returns an array of all the keys of any object as an argument.

    The Object.values() returns an array of all the object's value.

    The Object.entries() returns an array of key-value pairs.

    for(const [key,value] of Object.entries(superman)) { console.log(`${key}: ${value}`); }

    Adding Properties

    object.key = "value";

    Changing Properties

    object["old_value"] = "new_value";

    Removing Properties

    delete object.key

    Nested Objects

    Objects that contain other objects

    Accessed by either dot, bracket notation, or a mix

    An object literal can be passed as a parameter to a function.

    Namespacing

    It is used to avoid name duplication inside a program. An object is created to contain the method or properties needed and are called or accessed with object.method() code.

    JSON

    It is a string representation of the object literal notation. There are, however, a couple of key differences:

    • Property names must be double-quoted
    • Permitted values are double-quoted strings, numbers, true, false, null, arrays and objects
    • Functions are not permitted values

    The Math Object

    It is a built-in object that has several properties representing mathematical constants, as well as methods that carry out a number of common mathematical operations.

    • Math.PI - The ratio of the circumference and diameter of a circle
    • Math.SQRT2 - The square root of 2
    • Math.SQRT1_2 - The reciprocal of the square root of 2
    • Math.E - Euler's constant
    • Math.LN2 - The natural logarithm of 2
    • Math.LN10 - The natural logarithm of 10
    • Math.LOG2E - Log base 2 of Euler's constant
    • Math.LOG10E - Log base 10 of Euler's constant
    • Math.abs() - returns the absolute value of a number
    • Math.ceil() - rounds a number up to the next integer
    • Math.floor() - rounds a number down to the next integer
    • Math.round() - rounds a number to the nearest integer
    • Math.trunc() - returns the integer
    • Math.exp() - raises a number to the power of Euler’s constant
    • Math.pow(x, y) - raises any number (x) to the power of another number (y)
    • Math.sqrt() - returns the positive square root of a number
    • Math.cbrt() - returns the cube root of numbers
    • Math.hypot() - returns the square root of the sum of the squares of all its arguments. It can be used to calculate the hypotenuse of a right-angled triangle
    • Math.log() - returns the natural logarithm of a number (Math.log2() or
    • Math.log10())
    • Math.max() - returns the maximum number from its arguments
    • Math.min() - returns the minimum number from its arguments
    • Math.sin() - returns the sine of an angle
    • Math.cos() - returns the cosine of an angle
    • Math.tan() - returns the tangent of an angle
    • Math.asin() - returns the arcsine of a number. The result is an angle
    • Math.acos() - returns the arccosine of a number. The result is an angle
    • Math.atan() - returns the arctangent of a number. The result is an angle
    • Math.random() method is used to create random numbers

    Methods for the hyperbolic functions, sinh(), cosh() and tanh() were added, as well as their inverses asinh(), acosh() and atanh().

    Regular Expressions

    A regular expression (RegExp) is a pattern that can be used to search strings for matches to the pattern. A common use is 'find and replace' type operations.

    test() - tests if a string contains the pattern and provides a boolean answer

    exec() - returns an array containing the first match found

    Document Object Model (DOM)

    It represents an HTML document as a network of connected nodes that form a tree-like structure. The DOM treats everything on a web page as a node.

    • getElementById() - returns a reference to the element with a unique id attribute that is given as an argument
    • getElementsByTagName() - returns a live node list of all the elements with the tag name that is given as an argument
    • getElementsByClassName() - returns a live node list of all elements that have the class name that is supplied as an argument
